## ディレクトリ
コンテスト終了後に試行錯誤した場合は、afterRoomディレクトリを作成し、
ファイル名は、先頭に a をつける。
## 使い方
1. コンテスト名のディレクトリを作る
2. コードを書く
3. 以下のコードを実行して、移動させる
~~~
fish ./1next.sh
~~~
4. main.goに次の問題を解く
5. 以下のコードを実行して、pushまでやる
~~~
fish ./2cleaningUp.sh
~~~
6. おしまい
